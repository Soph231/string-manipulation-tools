# String Manipulation Tools

Sample files and exercises accompanying articles on string manipulation and text processing.

This repository contains small examples demonstrating how different tools can be used to clean, extract, and transform text data. Many of the examples are inspired by biological datasets, but the ideas are applicable to a wide range of disciplines.

The repository is intended to complement the accompanying articles rather than replace them. Each folder contains sample files and suggested tasks that allow readers to practise the techniques discussed in the articles.

## Topics Covered
### Google Sheets and Excel
* Splitting text into columns
* Replacing text
* Extracting parts of strings
* Identifying patterns in text
### Text Editors
* Find and replace
* Previewing changes
* Batch editing text
### Regular Expressions
* Basic regex syntax
* Combining patterns
* Common mistakes
* Additional learning resources
### Shell Scripting
* Automating repetitive tasks
* Batch file renaming
* Simple command-line workflows
